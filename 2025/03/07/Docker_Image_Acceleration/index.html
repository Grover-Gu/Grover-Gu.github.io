1. 修改Docker配置文件

编辑Docker对的配置文件，如果不存在可直接创建。配置文件地址 /etc/docker/daemon.json

2. 添加镜像加速地址

  • sudo tee：将内容写入文件，并需要管理员权限。

  • /etc/docker/daemon.json：Docker 的配置文件路径。

  • <<-'EOF'：使用 Here Document 的方式输入多行内容，EOF 是结束标记。

  • registry-mirrors：配置多个镜像加速地址。
